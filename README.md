# Server-side GTM Deployment Script

This Bash script automates the deployment of a **Server-side Google Tag Manager (sGTM)** container to **Google Cloud Run**, based on Simo Ahava’s original script.

It sets up both:
- **Production service** (`-prod`)  
- **Debug/Preview service** (`-preview`)
